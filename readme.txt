Surveille et gère les modifications de fichiers dans l'éditeur de WordPress, y compris les thèmes. **Simple Editor Control 3.0.0** - Le Guerrier de la Gestion de Code.

== Description ==

Décrit comme le "Spartiate du Code," **Simple Editor Control** n'est pas juste un plugin; c'est votre allié dans la bataille pour une gestion de fichiers précise et puissante dans WordPress. Avec la version 3.0.0, nous avons transcendé les limites de la simple surveillance pour vous offrir un contrôle absolu :

- **Édition Directe**: Une simple clique sur un fichier vous propulse dans l'éditeur, prêt à modifier.
- **Téléchargement Historique**: Revivez l'histoire de vos fichiers; téléchargez n'importe quelle version passée.
- **Refonte Visuelle**: Un design rehaussé, inspiré par la discipline spartiate, pour une clarté et une efficacité inégalées.
- **Édition et Téléchargement Intégrés**: Pas besoin de naviguer ailleurs; éditez et téléchargez directement depuis l'interface.
- **Design Redéfini**: Une interface utilisateur refondue pour une navigation fluide et une présentation visuelle impressionnante.
  - **Note Importante**: Le design modifié du `<body>` est **exclusif** à la page du plugin **Simple Editor Control**. Le reste de votre site WordPress ne sera pas affecté par ces changements visuels.
- **Améliorations de Performance**: Optimisations pour une rapidité de traitement des diffs qui ferait honneur à un hoplite.
- **Sécurité Renforcée**: Des mesures de sécurité additionnelles pour protéger vos modifications comme les remparts de Sparte.

== Installation ==

1. **Téléchargez** le plugin depuis WordPress.org.
2. **Installez** le via le gestionnaire de plugins de WordPress.
3. **Activez** le plugin.
4. **Accédez** à **Outils > Simple Editor Control** pour plonger dans la gestion de vos modifications.

== FAQ ==

### Comment puis-je visualiser les fichiers modifiés?

Accédez simplement à **Outils > Simple Editor Control** dans le tableau de bord de WordPress.

### Est-il compatible avec les thèmes WordPress?

Absolument, depuis la version 2.1.2, avec des améliorations successives.

### Pourquoi choisir Simple Editor Control?

Parce que vous recherchez la précision et la puissance dans la gestion des modifications, le débogage, et la maîtrise de versions. Simple Editor Control est l'arme ultime pour cela.

### Est-ce que le plugin est sécurisé?

Oui, nous respectons les standards de sécurité WordPress, et avec 3.0.0, nous avons ajouté encore plus de fortifications.

### Pourquoi mon site semble-t-il différent après avoir installé Simple Editor Control?

Le design du `<body>` change uniquement lorsque vous êtes sur la page du plugin **Simple Editor Control**. Le reste de votre site reste inchangé.
